# Student Performance Analysis

## Project Overview

This project analyzes student academic performance using the StudentsPerformance dataset.

## Objectives

- Explore and clean the dataset
- Analyze factors affecting student scores
- Identify at-risk students
- Create visualizations
- Provide recommendations for school administrators

## Tools Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Google Colab

## Key Findings

- Test preparation significantly improves performance.
- Higher parental education levels are associated with better scores.
- Reading and writing scores are strongly correlated.
- Female students perform better in reading and writing.
- Male students perform better in mathematics.
